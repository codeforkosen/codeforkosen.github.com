コードで、高専を変える

Code for KOSENは、 高専生活における「こうしたい！」を高専生が IT と デザイン の力で実現していくための組織です。

コードで、高専を繋ぐ

高専の抱える問題の多くは「共通」したものが多いでしょう。 Code for KOSENでは、高専間で抱える課題をオープンにし共有することが、 解決への糸口であると考えています。

コードで、高専を楽しもう

「ものづくり」を純粋に楽しみたい。 そんな高専生が全国にいるとしたら… ワクワクしませんか？ 小さなことからはじめましょう。それがあなたの Code for X です。
