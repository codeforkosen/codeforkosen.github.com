コードで、高専を変える

Code for KOSENは、 高専生活における「こうしたい！」を
高専生が IT と デザイン の力で実現していくための組織です。

コードで、高専を繋ぐ

高専の抱える問題の多くは「共通」したものが多いでしょう。 Code for KOSENでは、 高専間で抱える問題を共有することが、 解決への糸口であると考えています。

コードで、高専を楽しもう

「ものづくり」を純粋に楽しみたい。
そんな高専生が全国にいるとしたら… ワクワクしませんか？
小さなことからはじめましょう。それがあなたの Code for X です。

活動・事例

actif - 高専祭をてのひらに

年に一度の高専祭、ただ参加するだけじゃつまらない！ actif（あくてぃふ）とは、明石高専生による明石高専祭のためのSNS。参加者と運営者の双方を結びたい。そんなビジョンを実現するアプリです。

actif が 日本最大級のWebアプリ開発コンテスト「Mashup Award 9」にて U-23賞 を受賞しました。 Code for Kosen、MA9にてU-23賞受賞！プログラミングでより豊かな高専ライフ / 福野泰介の一日一創

今後の活動について

Code for KOSEN では、 一高専で行う小規模なイベントから、多高専を交えた大規模なイベントを企画していく予定です。
